# Final Whistle Frontend

Frontend for the Final Whistle football post-match recording product.

## Getting Started

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

Set `NEXT_PUBLIC_API_URL` if your backend is not running on `http://localhost:8080`.

## Validation

```bash
npm run lint
npm run type-check
npm run build
```

## Current Scope

- Public match, team, and player browsing
- Dev login with session cookie auth
- Match check-in create and edit flow
- Profile summary and check-in history

## Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS 4
